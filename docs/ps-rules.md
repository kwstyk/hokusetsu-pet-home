# 運用ルール（週次・月次タスク）

## 1. 週次タスク（目安：週 1 回・30〜60分）

| タスク                           | 詳細                                                                 | 所要時間目安 |
| -------------------------------- | -------------------------------------------------------------------- | ----------- |
| 物件の募集状況チェック           | 掲載中の物件（10〜20件）のステータス確認。募集終了していないか。      | 20〜30分    |
| `status` / `lastCheckedAt` 更新 | 変更がある物件のみ `listings.ts` を修正し、日付を更新。             | 10〜20分    |
| サイトのざっくり表示確認         | トップ /listings /contact /first-time を一通り開いて崩れがないか確認。 | 5〜10分     |

## 2. 月次タスク（目安：月 1 回・1〜2時間）

| タスク                      | 詳細                                                                 | 所要時間目安 |
| --------------------------- | -------------------------------------------------------------------- | ----------- |
| `/first-time` の見直し      | サービス内容・対応エリアが現状とズレていないか確認。必要なら微修正。 | 20〜30分    |
| `/about` の見直し          | 資格や経歴・連絡先などの情報を最新化。                                | 10〜20分    |
| `/cases` or `/blog` の追加 | 余裕があれば、成約事例 or ブログを 1 本追加。                          | 30〜60分    |

## 3. 不定期タスク

- 法改正・プライバシーポリシーの見直し
  - 新しい法規制が出た場合、`/privacy` `/security` を必要に応じて更新
- デザインの調整
  - ナビのラベル変更、CTA 文言の調整などは、必要に応じて少しずつ実施
